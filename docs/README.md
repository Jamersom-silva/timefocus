# ⏰ TimeFocus

Sistema de gerenciamento de tempo com foco em produtividade pessoal usando a técnica Pomodoro. Desenvolvido com **FastAPI** no backend e **React + TypeScript** no frontend.

## 🚀 Funcionalidades

- Timer Pomodoro com controle de ciclos
- Autenticação JWT (Login e Registro)
- Gamificação: conquistas e pontuação
- Geração de relatórios de produtividade (diário/semanal)
- Integração com frontend React
- API RESTful com FastAPI
- Testes com pytest

## 🧱 Tecnologias

- Python 3.11
- FastAPI
- Uvicorn
- Python-Jose (JWT)
- Passlib (hash de senhas)
- Pytest (testes)
- React + TypeScript (frontend)
